Network

Dual-port 10G upgrade

I added an Intel X550-T2 dual-port 10GBase-T NIC to the Proxmox host and split it into two dedicated lanes: storage and uplink.

One port is passed through directly to the TrueNAS VM for a point-to-point storage network to my workstation. The other stays host-owned and is bridged to the core services VM.

Storage lane

10GbE point-to-point

Dedicated NAS ↔ workstation link on its own /30 subnet (no gateway). Measured at ~9.3 Gbit/s with iperf3.

Uplink lane

Core services uplink

Second port is bridged to the core VM so containers and updates don’t fight the management interface.

Stability

No more renumbering

Interface names are pinned by MAC on the host so adding PCIe cards doesn’t shuffle NIC names and take the cluster down.
